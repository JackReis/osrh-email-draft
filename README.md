# OSRH Email Builder

Interactive single-page builder for the **Vista Del Mar / Our Sea Ranch Home** prior-guest
re-engagement email. Pick the preferred wording for each section, preview it live, and copy
the finished email (HTML or plain text) into MailChimp.

- Live: https://osrh-email-draft.jkre.is
- Audience: past guests only ("Prior Guests" MailChimp segment), approval-gated send.
- No recipient data is included in this repo — it builds copy only.

Deploy: GitHub Pages (branch `main`, root), custom domain via `CNAME`, Cloudflare-proxied.
